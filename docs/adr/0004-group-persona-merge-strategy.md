# Group Persona uses minimum for constraints, average for preferences

When merging member Personas into a Merged Persona, not all dimensions behave the same way. Physical constraints and comfort floors have hard ceilings: a group cannot hike faster than its least-fit member, and a group cannot visit a spot that violates the most culturally conservative member's threshold. These dimensions (Stamina, Solitude Need, Cultural Affinity) use the minimum across all members. Style preferences with no hard floor (Curiosity, Nature Affinity) use the average — a reasonable middle ground that doesn't punish any one member. A pure average across all dimensions would produce a Stamina level that exhausts the weakest member; a pure minimum would produce a trip that bores the most adventurous.
